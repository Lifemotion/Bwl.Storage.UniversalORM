Основной модуль UniversalORM
1) CommonObjStorage - реализует интерфейс IObjStorage
2) SearchParams - класссы в данной папке предназначены для задания параметров поиска.
3) MemoryStorage - класс для работы с хранилищем данных в памяти(обновление/сохранение/удаление и т.п.)
4) MemoryStorageManager - реализует интерфейс IObjManager. Создает менеджер для работы с памятью(сохранение/удаление/замена и т.п.)
5) ObjContainer - реализует интерфейс IObjBase. Хранит объект, его ID и тип. Предназначен для индексации обектов.
